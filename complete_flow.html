Create a Checkout Preference

First you need to create a checkout preference with all payment related information in your server: 

              //Create an Item
              Item item = new Item("itemId", 1);

              //Create a CheckoutIntent with an access token to your server and the item
              CheckoutIntent checkoutIntent = new CheckoutIntent(MERCHANT_ACCESS_TOKEN, item);

              //Finally, create a preference
              MerchantServer.createPreference("http://merchanturl.com", "/preferences",checkoutIntent, new Callback<CheckoutPreference>() {
              @Override
              public void success(CheckoutPreference checkoutPreference, Response response) {
                  //Congrats, you have a preference!
              }

              @Override
              public void failure(RetrofitError error) {
                  //Handle failure
              }
            });

The preference payment methods will exclude those payment methods or types you don't want to display in your app. This property is not mandatory. You can check our payment methods and payment types availables for more information.

Start checkout flow with a preference

              new MercadoPago.StartActivityBuilder()
                        .setActivity(activity)
                        .setPublicKey(yourPublicKey)
                        .setCheckoutPreference(preference)
                        .setShowBankDeals(true)
                        .startCheckoutActivity();
