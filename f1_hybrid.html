Personalizá tu checkout

El primer paso es agregar la SDK a tu proyecto. Descargá la SDK de MercadoPago y agregá la siguiente linea dónde quiereas integrarnos. 

                        
                               <script src="js/MercadoPagoHibrido.js"></script>

El segundo paso es indicarnos tu clave pública para que cada vez que llamemos a un servicio lo hagamos en tu nombre. 

                        
                               MercadoPagoService.setPublicKey("TEST-ad365c37-8012-4014-84f5-6c895b3f8e0a");

Medios de pago

Es la entidad participante del pago en sí misma. Desde la marca de la tarjeta, pasando por la entidad de cobro en efectivo hasta el banco en el que se encuentra el cajero automático para que el usuario realice el pago.
Si el medio de pago pertenece a una tarjeta te brindamos la información que necesitás para armar tu formulario de tarjeta y sus validaciones, por ejemplo la longitud del número de tarjeta, si el código de seguridad está al frente o al dorso de la tarjeta, su longitud, entre otros. El medio de pago puede ser offline. En este caso te indicaremos, entre otros, el tiempo de acreditación ya que el pago quedará en un estado pendiente que cambiará cuando el usuario se acerque a la entidad correspondiente a finalizar el pago en efectivo.

*Probá un ejemplo de respuesta de la API de medios de pago (/v1/payment_methods).

El siguiente código hace el llamado a la API y convierte la respuesta en un listado de PaymentMethod. 

                        
MercadoPagoService.getPaymentMethods().get(function(response){
   // Listo, armá tu UI para el listado de medios de pago
   $scope.paymentMethods = response;
},function(error){
   // UPS! Algo salió mal.
});

Bancos

El banco es la entidad que emite la tarjeta. Como procesadores de pagos establecemos promociones con los distintos bancos para que tus usuarios puedan tener mayor flexibilidad a la hora de comprar. La marca de la tarjeta (payment_method) que esté usando tu usuario puede ser emitida por más de un banco. Para ofrecerle las promociones correctas, necesitamos que seleccione el banco que emitió su tarjeta. Si el banco no fuera seleccionado o eligiera uno incorrecto, el pago no será procesado.

*Probá un ejemplo de respuesta de la API de bancos para un medio de pago seleccionado (/v1/payment_methods/card_issuers).

En el siguiente código te mostramos cómo podes hacer el llamado al servicio de Issuers dado un medio de pago desde nuestra SDK. 

                        
var paymentMethodId = "master";
var bin = ""; // Primeros seis dígitos de la tarjeta (si los conocieras)

MercadoPagoService.getIssuers(paymentMethodId, bin)
   .get(function(response) {
      // Listo, armá tu UI para el listado de bancos
      $scope.cardIssuers = response;
   },function(error){   
      // UPS! Algo salió mal.
   }
);

Cuotas

Una vez que tu usuario seleccione el medio de pago y el banco que emitió su tarjeta, es importante que, para ofrecerle mayor flexibilidad al momento de pagar, le permitas seleccionar la cantidad de cuotas en la que va a realizar el pago.
Sólo te pedimos que nos indiques el medio de pago que seleccionó el usuario, el banco y, de tenerlo, el BIN de la tarjeta (dependiendo cómo pensaste tu experiencia de pagos). Qué es el BIN? Son los primeros seis dígitos de la tarjeta, con el que desde nuestros servicios podríamos saber a qué medio de pago pertenece y qué banco emitió la tarjeta.
El monto es opcional pero permite obtener una descripción recomendada para la cuota, lo que es muy práctico si se quiere mostrar un listado de cuotas escribiendo muy pocas líneas de código.

*Probá un ejemplo de respuesta de la API de cuotas para el monto que deseas cobrar y el medio de pago y banco seleccionados (/v1/payment_methods/installments).

El siguiente código hace el llamado a la API enviando los parámetros mecionados y convierte la respuesta en un listado de Installment. 

                        
// Medio de pago seleccionado
var paymentMethodId = "master";

// Banco seleccionado en el paso anterior
var issuerId = "326";

// Monto que vas a cobrar
var amount = 100.0;

MercadoPagoService.getInstallments(paymentMethodId, issuerId, amount)
   .get(function(response) {
      // Listo, armá tu UI para el listado de cuotas
      $scope.installments = response[0]["payer_costs"];
   },function(error){
      // UPS! Algo salió mal.
   }
);

Tipos de documento

Debes recolectar el tipo y número de documento del portador de la tarjeta. Dependiendo las regulaciones de cada país en el que procesamos, los bancos y las marcas de tarjetas nos solicitan estos datos para efectuar pagos. Tenemos un servicio que, dependiendo el país en el que operes, te devuelve el listado de tipos disponibles con sus respectivas configuraciones como nombre, tipo de dato (numérico o alfanumérico), longitud máxima y mínima, etc.

*Probá un ejemplo de respuesta de la API de tipos de documento (/v1/identification_types).

En el siguiente código te mostramos cómo podes hacer el llamado al servicio de IdentificationTypes. 

                        
MercadoPagoService.getIdentificationTypes()
   .get(function(response) {
      // Listo, armá tu UI para el listado de tipos de documento
      $scope.identification_types = response;

Promociones

Hablamos con las marcas de las tarjetas y con los bancos para poder brindarle a tus usuarios las mejores promociones a la hora de pagar. Consideramos que es muy importante que en tu flujo de pagos le cuentes a tus usuarios las comodidades que tiene para abonar, ya que es más probable que elija un medio de pago conveniente para él y pueda concretar el pago.

Crear preferencias

TEXTO 

                        
var base_url = "http://api.tunombre.com";
var create_preference_uri = "/create_preference";

var item_id = "item_id";
var item_quantity = 1;
var item_unit_price = 100;

// Armá un mapa con los datos de la compra y el mail de tu cliente.
var preference_map = {
   "item_id": item_id,
   "amount": item_unit_price * item_quantity,
   "payer_email": "EMAIL_DE_TU_USUARIO",
};

// Envia la información a tu servidor
MercadoPagoService.createPreference(base_url, create_preference_uri, preference_map)
   .save(function(response) {
      // La preferencia se creó correctamente.
      // Aquí puedes iniciar el Checkout de MercadoPago.
         MercadoPagoService.setPublicKey("TEST-ad365c37-8012-4014-84f5-6c895b3f8e0a");
         MercadoPagoService.setPrefId(response.id);
         MercadoPagoService.startCheckout(function(payment){
            console.log(payment);
         });
   }, function(error) {
      // Ups, ocurrió un error
   }
);
